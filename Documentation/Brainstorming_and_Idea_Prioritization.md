## 1. Introduction

Brainstorming is an important step in project development because it helps students and researchers explore different ideas before choosing the most suitable one. A well-planned brainstorming process allows a project team to compare options based on relevance, feasibility, data availability, and practical impact. It also reduces the risk of selecting a topic that is too difficult, too simple, or not useful for real-world applications.

For academic projects, brainstorming is especially valuable because it encourages critical thinking and helps students connect technical learning with practical problem-solving. In this project, brainstorming was used to compare several machine learning ideas and select the most appropriate one for implementation, analysis, and presentation.

## 4. Final Selected Idea

The final selected topic is Human Development Index (HDI) Prediction Using Machine Learning.

This idea was chosen because it offers a balanced combination of academic value, technical learning, and social relevance. The project is not only suitable for machine learning practice but also meaningful in understanding how data can support development and policy-related decision-making.

### Reasons for Selection

- Real-world relevance: HDI is a widely recognized measure of development and social progress.
- Availability of structured dataset: The dataset is available in a clear tabular form, making it suitable for supervised learning.
- Suitable for Linear Regression: The problem can be effectively modeled using linear regression due to the relationship between input features and the target variable.
- Educational value: The project helps students understand data preprocessing, model training, evaluation, and interpretation.
- Opportunity to learn Flask deployment: The model can be deployed using Flask, which is useful for building a web-based application.
- Suitable for GitHub portfolio: The project is complete, practical, and visually presentable for future job and academic opportunities.
